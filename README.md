# Scudo Chiamate — Documenti legali

Repository pubblico per hostare le pagine legali dell'app **Scudo Chiamate**:

- [Informativa sulla Privacy](https://orazio84-bit.github.io/scudochiamate-legal/privacy.html)

Le pagine sono servite tramite GitHub Pages dal branch `main`.

## Aggiornamenti

Per modificare la privacy policy, edita `privacy.html`, aggiorna la data in alto e committa. Il deploy GitHub Pages è automatico (~1 minuto).
